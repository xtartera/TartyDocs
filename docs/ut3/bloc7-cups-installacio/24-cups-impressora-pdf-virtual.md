# :material-file-pdf-box: CUPS – Impressora PDF virtual

!!! abstract "Concepte clau"
    El paquet `cups-pdf` crea una impressora virtual que genera fitxers PDF. Els documents enviats a aquesta impressora es desen a `/var/spool/cups-pdf/<usuari>/`, útil per a proves i entorns sense impressora física.
