# :material-console: CUPS – Ordres d'impressió (lp, lpq, cancel)

!!! abstract "Concepte clau"
    Les ordres bàsiques de CUPS: `lp -d impressora fitxer` (enviar treball), `lpq` (veure la cua), `lpstat -p` (estat de les impressores) i `cancel IDtreball` (cancel·lar un treball en cua).
