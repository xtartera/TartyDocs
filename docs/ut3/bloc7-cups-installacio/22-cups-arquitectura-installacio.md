# :material-printer: CUPS – Arquitectura i instal·lació

!!! abstract "Concepte clau"
    CUPS (Common UNIX Printing System) és el sistema d'impressió estàndard a Linux. Actua com a intermediari entre les aplicacions i les impressores, oferint una interfície web al port 631 i el protocol IPP.
