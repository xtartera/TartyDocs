# :material-ip-network: NFS – Control d'accés per IP

!!! abstract "Concepte clau"
    NFS restringeix l'accés per adreça IP directament a `/etc/exports`: cada exportació especifica quines IPs o rangs de xarxa hi poden accedir i amb quines opcions.
