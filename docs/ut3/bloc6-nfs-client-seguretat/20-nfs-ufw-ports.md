# :material-wall-fire: NFS – UFW i ports de servei

!!! abstract "Concepte clau"
    NFS usa els ports 111 (rpcbind) i 2049 (nfs). Per permetre l'accés des de clients específics amb UFW cal obrir ambdós ports per IP o rang de xarxa amb `ufw allow from`.
