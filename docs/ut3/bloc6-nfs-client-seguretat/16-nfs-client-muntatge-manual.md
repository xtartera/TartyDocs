# :material-folder-sync: NFS Client – Muntatge manual

!!! abstract "Concepte clau"
    El client NFS usa `mount -t nfs servidor:/ruta /punt-de-muntatge` per accedir a una exportació remota. El paquet `nfs-common` és necessari al client per usar el protocol NFS.
