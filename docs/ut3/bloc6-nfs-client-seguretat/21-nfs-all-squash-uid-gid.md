# :material-account-convert: NFS – all_squash, anonuid, anongid

!!! abstract "Concepte clau"
    `all_squash` mapeja tots els usuaris del client a un usuari anònim al servidor. `anonuid` i `anongid` defineixen l'UID/GID d'aquest usuari anònim, permetent controlar qui és el propietari efectiu dels fitxers creats via NFS.
