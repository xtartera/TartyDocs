# :material-shield-lock: NFS – Opcions de seguretat (noexec, nosuid)

!!! abstract "Concepte clau"
    Les opcions de muntatge `noexec` (no permet executar binaris) i `nosuid` (ignora els bits SUID/SGID) limiten el risc de seguretat en exportacions NFS accessibles per usuaris poc de confiança.
