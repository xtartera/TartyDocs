# :material-file-cog: NFS – Muntatge persistent (/etc/fstab)

!!! abstract "Concepte clau"
    Per muntar automàticament una exportació NFS a cada reinici, s'afegeix una entrada a `/etc/fstab` amb el tipus `nfs` i l'opció `_netdev` (munta quan la xarxa estigui disponible).
