# :material-shield-account: CUPS – Restriccions per grups (cupsd.conf)

!!! abstract "Concepte clau"
    `cupsd.conf` controla qui pot administrar CUPS i usar les impressores. La directiva `AllowGroup lpadmin` dins d'un bloc `<Location>` restringeix l'accés a membres d'un grup Linux concret.
