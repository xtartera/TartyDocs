# :material-microsoft-windows: CUPS + Samba – Compartició amb Windows

!!! abstract "Concepte clau"
    Combinant CUPS i Samba (`printcap name = cups`, secció `[printers]` a `smb.conf`) es poden compartir impressores CUPS amb clients Windows, que les veuen com a impressores de xarxa Windows estàndard.
