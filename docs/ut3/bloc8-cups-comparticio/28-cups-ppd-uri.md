# :material-file-document-outline: CUPS – PPD i URI d'impressora

!!! abstract "Concepte clau"
    Els fitxers PPD (PostScript Printer Description) descriuen les capacitats d'una impressora. La URI identifica el dispositiu d'impressió: `cups-pdf:/` per a PDF virtual, `ipp://servidor/printers/nom` per a impressió remota.
